# How to Run This Project

1. Clone the repository:
   ```bash
   git@github.com:hedgehogoutfit/TreeMenu.git
   
   ```

2. Run setup:
   ```bash
   ./setup.sh
   ```

3. Access the project:
   - Website: http://localhost:8000
   - Admin: http://localhost:8000/admin
     - Username: `admin`
     - Password: `admin123`
