# How to Run This Project

1. Clone the repository:
   ```bash
   git clone https://github.com/your/repo.git
   cd your_project
   ```

2. Run setup:
   ```bash
   ./setup.sh
   ```

3. Access the project:
   - Website: http://localhost:8000
   - Admin: http://localhost:8000/admin
     - Username: `admin`
     - Password: `admin123`
